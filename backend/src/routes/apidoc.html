One-Qr-Gen API Documentation

Welcome! Below are the available endpoints for QR code generation:

POST /v1/qrgen/account

Generate a QR code for a bank account.

Body: { accountNumber, ifsc, name }

POST /v1/qrgen/upi

Generate a QR code for a UPI ID.

Body: { upiId, name, amount? }

POST /v1/qrgen/url

Generate a QR code for a URL.

Body: { url }

POST /v1/qrgen/vcard

Generate a QR code for a vCard contact.

Body: { name, phone, email, ... }

All endpoints return a QR code image or SVG.
